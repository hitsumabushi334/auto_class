# 海外エンジニアが最近使い始めたAIプロンプトツール【Repo Prompt】を解説します！

## 全体要約

この動画では、海外エンジニアの間で注目されているAIプロンプトツール「Repo Prompt」が紹介されています。AIによるコード修正で起こりやすいハルシネーション（誤った情報生成）を防ぎ、大規模なコードベースでも正確かつ効率的にAIを活用することを目的として開発されました。Mac版は現在無料でダウンロード可能で、Windows/Linux版はウェイティングリストに登録できます。動画では、実際にローカルにあるウェブサイトのコードを読み込み、マウスカーソルアニメーションがページ全体で機能するように修正するデモが行われています。Repo Promptを使うことで、リポジトリのファイル構造をAIに正確に伝え、XML形式で構造化されたプロンプトを生成・適用できるため、AIの応答精度が向上します。無料版で利用できる機能と、有料のPro版で追加される機能（ファイル自動選択、コードベース解析、カスタムAIモデル利用など）についても説明されています。

### Repo Promptの概要と開発された目的


キーワード:

- Repo Prompt
- AIプロンプトツール
- AIコーディング
- ハルシネーション
- リポジトリ
- コード修正

要約:

Repo Promptは、AIを利用したコード修正において、特に大規模なコードベースで発生しがちなハルシネーション問題を解決するために開発されたAIプロンプトツールです。リポジトリ全体のファイル構造や依存関係をAIに正確に伝えることで、より適切で信頼性の高いコード修正の生成を支援することを目的としています。


ポイント:

- AIによるコード修正の課題（ハルシネーション等）を解決するツール。
- コードベース全体の情報をAIに正確に伝えることを支援。
- 大規模なプロジェクトでのAI活用効率を高める。
- Mac版はダウンロード可能、Windows/Linux版はウェイティングリスト。

専門用語:

- ハルシネーション : AIが事実に基づかない、誤った情報やコードを生成すること。
- リポジトリ : プロジェクトのソースコードや関連ファイルを管理する場所（フォルダやバージョン管理システム）。

### Repo Promptを使ったコード修正のデモンストレーション


キーワード:

- デモ
- GUI
- ファイル選択
- プロンプト生成
- XML
- AI Studio
- Gemini
- Diff
- 修正適用
- マウスカーソルアニメーション

要約:

動画では、ローカルにあるウェブサイトのコードをRepo Promptで開き、マウスカーソルアニメーションがページ全体で動作しない問題を修正するデモが行われました。まず、対象のコードファイル（HTMLとJS）を選択し、修正内容をプロンプトとして入力します。Repo Promptは選択されたファイルを分析し、AIが理解しやすいXML形式のプロンプトを生成します。このXMLプロンプトをGoogle AI Studio（Gemini）に送信し、生成された修正を含むXML応答をRepo Promptに戻します。Repo Promptの「Review Changes」機能でAIによる修正内容（Diff形式）を確認し、問題なければ「Accept All Files」でファイルに適用します。修正後、ウェブサイトをリロードすると、アニメーションがページ全体で正常に動作することが確認できました。


ポイント:

- ローカルのコードファイルをツールに読み込む。
- 修正したい内容を自然言語でプロンプトとして入力。
- AIが処理しやすいXML形式に自動変換される。
- 生成されたXMLをAIに送信し、応答を受け取る。
- AIの修正内容をDiff形式で視覚的に確認できる。
- 修正を簡単にファイルに適用できる。
- 特定の課題（アニメーションの動作範囲）が解決された事例を示す。

専門用語:

- GUI : グラフィカルユーザーインターフェース。マウスなどで操作できる画面表示。
- XML : データの構造を記述するためのマークアップ言語。ここではAIとのやり取りに用いられる。
- Diff : ファイル間の差分を表示する形式。コードの追加、変更、削除箇所が分かりやすく表示される。

### Repo Promptの料金体系とPro版の機能


キーワード:

- 料金
- 無料版
- Pro版
- サブスクリプション
- Lifetime access
- Context Builder
- CodeMaps
- API連携
- AIモデル

要約:

Repo Promptには、基本的な機能が使える無料版（Free Tier）があります。さらに、より高度な機能を利用できる有料のPro Upgrade Optionsが提供されています。有料版には月額・年額のサブスクリプションと買い切り型のLifetime Accessがあります。Pro版では、AIとのやり取りにおけるトークン数の制限がなくなり、作業に必要なファイルをAIが自動で選択するContext Builder機能（要ライセンス）や、コードベース全体をスキャンして構造や参照を把握するCodeMaps機能などが利用可能になります。また、OpenAI、Anthropic、Geminiなど複数のAIモデルやAPIキーを設定して利用できる柔軟性も提供されます。


ポイント:

- 無料版で基本的なコードプロンプト機能を利用可能。
- 有料版でトークン制限解除やファイル自動選択機能が追加。
- 複数のAIモデル（OpenAI, Gemini等）やAPIキーを利用可能。
- 買い切り型のオプションもある。
